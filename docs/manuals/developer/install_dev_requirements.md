# Required Tools

Python: https://www.python.org/downloads/  
Node.js (includes npm): https://nodejs.org/  
Make: https://www.gnu.org/software/make/

Install (Ubuntu/Debian):
```bash
sudo apt update
sudo apt install -y python3 python3-venv nodejs npm make
```

Confirm:
```bash
python --version
node --version
npm --version
make --version
```
